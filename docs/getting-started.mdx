title: Getting Started Adding Command K
---

## Install Command K

```js
npm install commandk
```

## Add it to a page

### Vanilla html+javascript

```html
<html>
  <head>
    ...
    <script type="module" src="/build/commandk.esm.js"></script>
  </head>
  <body>
    <command-k />
  </body>
</html>
```

### React

```js
// add commandk somewhere near the root
import CommandK from "commandk-react";

const Root = () => {
  const commandkConfig = {
    token: "my-token"
  }

  return (
    <div>
      <CommandK config={commandkConfig} />
    </div>
  )
}
```

## Launching it other than with cmd+k or ctrl+k

It is highly recommended to place a button somewhere on the page that displays the hot key used to launch command k. The button itself should of course launch commandk as well:

```html
<script>
function openHandler() {
  let open = new Event('commandk:open');
  window.dispatchEvent(open);
}
</script>

<button type="button" onClick={openHandler}>Search everything (CMD+K)</button>
```
